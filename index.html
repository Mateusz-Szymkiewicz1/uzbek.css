Uzbek.css
Lightweight css library straight from central asia.  

About

Uzbek.css is a CSS library designed for building fresh, retro websites. It comes with several utility classes and components designed to do the heavy-lifting for you. It is free to use in your personal projects, we use MIT license. You can follow us on Github. Feel free to contribute to the project. To learn how to use uzbek.css visit our documentation
